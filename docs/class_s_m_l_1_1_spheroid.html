Detailed Description

Spheroid geometry.

A spheroid is an ellipsoid with two equal semi-diamters.

A spheroid is defined by a point (origin) and two radii:

  • Equatorial radius (axisXY)
  • Polar radius (axisZ)

Parametric description = (x^2 + y^2) / a^2 + z^2 / c^2 = 1
