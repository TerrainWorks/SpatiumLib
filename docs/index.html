Spatial Math Library (SML)

SML is a small header-only C++ library for geometrical operations.

Content

Matrix classes:

  • Matrix
  • Matrix4x4
  • Vector3
  • Point3

Geometry classes:

  • Plane
  • Sphere
  • Ellipsoid
  • Spheroid
  • OblateSpheroid - Useful for representing the earth

Usage

Simple copy the SML directory to your project. Then include the desired header files.
